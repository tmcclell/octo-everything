## 📊 Dashboards Built

### 1. Main App (`app.py`)
- Entry point with navigation
- Overview of SPACE and Copilot dashboards
- Data source indicator

### 2. SPACE Dashboard (`pages/1_space_dashboard.py`)
**Metrics Visualized:**
- **PR Cycle Time Trends**: Line chart showing median and mean cycle time over time
- **Review Turnaround Distribution**: Box plot by repository
- **Rework Rate Trends**: Bar chart with 3-week moving average
- **Work in Progress**: Stacked area chart of open PRs per repo

**Features:**
- Date range filter
- Repository multi-select filter
- KPI summary cards (Median Cycle Time, Review Time, Rework Rate, Avg WIP)
- Interactive Plotly charts
- Data refresh timestamp

### 3. Copilot Dashboard (`pages/2_copilot_dashboard.py`)
**Metrics Visualized:**
- **Active Users Trends**: Line chart of daily active/engaged users
- **Acceptance Rate Trends**: Scatter plot with 7-day moving average
- **Seat Utilization**: Line chart + gauge indicator
- **ROI Analysis**: Cumulative value vs. cost chart

**Features:**
- Date range filter
- ROI parameter controls (hourly rate, seat cost)
- KPI summary cards (Active Users, Acceptance Rate, Seat Utilization, ROI Ratio)
- Time & cost savings calculator
- Interactive Plotly charts

## 🚀 Running the Dashboard

### Prerequisites
```bash
pip install -r requirements.txt
```

### Launch Options

**Option 1: Using the run script**
```bash
cd devmetrics
python run_dashboard.py
```

**Option 2: Direct Streamlit command**
```bash
cd devmetrics
streamlit run app.py
```

**Option 3: From root directory**
```bash
streamlit run devmetrics/app.py
```

The dashboard will open at `http://localhost:8501`

**Key Features:**
- Responsive wide layout
- Interactive filters with immediate updates
- Cached data loading for performance
- Color-coded KPI deltas (improving/worsening trends)

- Mobile-friendly responsive design

## 🎯 Design Decisions

1. **Plotly over Matplotlib**: Interactive charts, better UX
2. **JSON Store Layer**: Decouples data source from UI - easy to swap dummy → live API
3. **Date/Repo Filters**: Essential for large teams to focus analysis
4. **KPI Cards First**: Quick overview before diving into charts
5. **Moving Averages**: Smooth out noise in daily metrics
6. **ROI Calculator**: Configurable parameters for different team contexts
7. **Gauge Chart for Utilization**: Visual target indicator (80% threshold)
8. **Cumulative ROI**: Shows value accumulation over time
